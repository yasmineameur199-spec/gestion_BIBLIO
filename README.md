# Projet de gestion de bibliothèque en ligne
## 1. Contexte du projet

Ce projet consiste en la conception et le développement d’une application web de gestion de bibliothèque en ligne.
Il a été réalisé dans un contexte académique, avec pour objectif principal de mettre en pratique les bases du développement web frontend, de la structuration d’un projet JavaScript, et de la collaboration via Git et GitHub.

L’application permet de gérer les éléments essentiels d’une bibliothèque (livres, interactions utilisateur) à travers une interface web simple, claire et maintenable.
Une attention particulière a été portée à la qualité du code, au respect des bonnes pratiques JavaScript, et à la séparation des responsabilités entre les fichiers HTML, CSS et JavaScript.

## 2. Objectifs du projet

Les objectifs du projet sont les suivants :

Concevoir une application web fonctionnelle de gestion de bibliothèque

Mettre en œuvre une structure de projet claire et professionnelle

Appliquer les bonnes pratiques JavaScript modernes :

fonctions fléchées

renommage explicite des variables

principes de modularité

Améliorer la lisibilité et la maintenabilité du code

Utiliser GitHub comme outil de collaboration et de suivi des versions

Corriger et optimiser progressivement le code via des commits successifs

## 3. Méthodologie et approche utilisée
Approche de développement

Le projet a été développé de manière itérative, avec des améliorations continues apportées au code à chaque étape.

L’approche suivie comprend :

Création d’une structure de projet claire dès le départ

Séparation stricte des fichiers :

HTML → structure

CSS → mise en forme

JavaScript → logique métier

Refactorisation progressive du code pour :

améliorer la lisibilité

respecter les standards modernes

corriger les erreurs initiales

Outils et technologies

HTML5

CSS3

JavaScript (ES6+)

Git / GitHub pour la gestion de versions et le travail collaboratif
